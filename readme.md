# Payprev Hiring Challenge

## Configuração

No arquivo `config.json` adicione seu banco de dados e o secret

```
{
  "connectionString": "ADICIONE O LINK DO SEU BANCO DE DADOS",
  "secret": "JWT SECRET"
}
```
